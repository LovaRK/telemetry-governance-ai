## Demo Script (Using This Implementation)

**Customer sees**:

1. **Executive Overview Dashboard**
   - 11 metrics with ⓘ icons
   - Provenance badges showing source and timestamp
   - MITRE/Lantern with baseline badges

2. **Customer asks**: "How is ROI Score calculated?"
   - Click ⓘ on ROI Score
   - Modal shows: `avg(composite_score) across all sourcetypes`
   - Shows component values (Util, Det, Qual for each sourcetype)
   - Shows result: 52.3

3. **Customer asks**: "Where does this data come from?"
   - ProvenanceBadge shows: "executive_kpis, Pipeline run 3b055b..., Generated 2 min ago"
   - Classification badge shows: "REAL" (data-backed)

4. **Customer asks**: "What about MITRE?"
   - Baseline badge visible with disclosure: "Demo baseline values"
   - Click → Shows: "Production Integration Pending"
   - Clear that this is reference data, not real integration

5. **Customer asks**: "Can I drill into the Tier 1 spend?"
   - (Out of scope for Wave 1)
   - Answer: "We'll add detailed drill-down in the next release. For now, you can see the breakdown in the Executive KPIs"
