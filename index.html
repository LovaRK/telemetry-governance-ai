Troubleshooting

Windows: "Smart App Control blocked this file"

Right-click Install datasensAI.bat → Properties → at the bottom tick Unblock → Apply → OK. Then run as Administrator again.

Or paste this in an Administrator PowerShell:

Unblock-File -Path ".\Install datasensAI.bat"
Windows: "Port 3002 already in use"

A previous install left containers running. Run the installer again (as Administrator) and press 7 to Uninstall, then press 1 for a fresh install.

Mac: "Apple cannot verify the developer"

Right-click Install datasensAI.command → Open → click Open in the popup. You only need to do this once.

Docker not starting

The installer opens Docker automatically. If it stalls:

  1. Fully quit Docker Desktop from the system tray / menu bar
  2. Reopen Docker Desktop manually and wait for the whale icon to stop animating
  3. Run the installer again — it will resume from where it stopped
Something else went wrong — run the health check

Windows (Admin PowerShell):

powershell %USERPROFILE%\datasensai\scripts\install\doctor.ps1

Mac (Terminal):

bash ~/datasensai/scripts/install/doctor.sh

The doctor script checks every component and prints exactly what is broken.

Full uninstall — remove everything

Inside the tools/ folder of the ZIP:

  • Windows: Run tools\uninstall_datasensAI_windows.ps1 as Administrator
  • Mac: Open tools/uninstall_datasensAI_mac.sh

This removes datasensAI, Docker Desktop, and the Ollama model cache (~5 GB freed).
